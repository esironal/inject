inject 
------

Yet Another JS asynchronous script loader

How to use
==========

       inject('moo.js').then('foo.js').then('bar.js').ready(function(){

           //moo.js & foo.js & bar.js ready
           //do something
       }) 

Browser support
---------------

       *IE6+
       *Chrome
       *Firefox 3+
       *Opera
       *Safari 4+
